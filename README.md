# EA Modeller

Enterprise Architecture Modeling Tool (ArchiMate® 3.2) – MVP.

## Getting started

```bash
npm install
npm run dev
```

## Useful scripts

- `npm run test`
- `npm run lint`
- `npm run format`
- `npm run build`


## GitHub Pages deployment

This repo includes a workflow that builds and deploys the app to GitHub Pages on pushes to `main`.

1. In GitHub: **Settings → Pages**
2. Under **Build and deployment**, ensure **Source** is set to **GitHub Actions**.

After the workflow runs, the site will be available at `https://<user>.github.io/pwa-modeller/`.
